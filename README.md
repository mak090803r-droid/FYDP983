
PROJECT PROPOSAL FORM 

1.  Title: Smart Glasses for Real Time Foreign Language Reading/ Translation

2. Description: 
This project aims to develop smart glasses capable of capturing printed or digital text, including books, foreign-language instructions, labels, and technical documents, translating them into a chosen language, and reading them aloud in real time. A lightweight camera mounted on the glasses captures the text, which is then processed using Optical Character Recognition (OCR) to convert it into digital form. AI-based algorithms convert the text into the target language, and the translated content is delivered audibly through built in speakers or earphones. This system provides real-time reading and listening support, enhancing accessibility, learning, and comprehension of foreign language content, technical manuals, and written instructions.
3. Expected Deliverables:
(a) AI module for OCR-based text extraction.
(b) AI-based translation system for multiple languages.
(c) Audio output to present translated content in real time.
(d)Smart glasses with camera integration for text capture.
(e) Lab demonstration reading and translating sample foreign-language text.
